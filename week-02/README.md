# Week 02: The Skeleton (Semantic HTML5)

## 🎯 Objective
Last week, we designed "Erasmus Brew" in Figma. This week, we build the **Skeleton**. You will translate your visual wireframe into a structured HTML document using semantic tags.

---

## 🛠 Task 1: Environment Setup
We will use **GitHub Codespaces** so you don't have to install anything on your computer.

1.  Open your GitHub repository in the browser.
2.  Click the green **"<> Code"** button -> **Codespaces** -> **Create codespace on main**.
3.  Inside the editor, create a new folder called `week-02`.
4.  Inside that folder, create a new file named `index.html`.
5.  **Install the Extension:** - Click the **Extensions** icon (the four squares on the left sidebar).
    - Search for **"Live Preview"** by Microsoft.
    - Click **Install**.

---

## 💻 Task 2: Building the "Erasmus Brew" Skeleton
Your goal is to recreate your Figma wireframe using only HTML tags. No colors or styling yet!

### 1. The Boilerplate
Type `!` and press `Tab` inside `index.html` to generate the basic structure. Change the `<title>` to `Erasmus Brew`.

### 2. Semantic Mapping
| Figma Element | HTML Tag | Description |
| :--- | :--- | :--- |
| **Shop Name** | `<header>` & `<h1>` | The very top of your page. |
| **Hero Image** | `<img>` | Use a placeholder from Unsplash. |
| **"About Us"** | `<section>` & `<p>` | A container for your shop's description. |
| **"Menu" List** | `<ul>` & `<li>` | An unordered list for Coffee, Tea, and Cake. |
| **Footer** | `<footer>` | Your shop's address and contact info. |

---

## 👁️ Task 3: Preview Your Work
Don't code in the dark! 

1.  Right-click your `index.html` file in the left-hand explorer.
2.  Select **"Show Preview"** (this is enabled by the Live Preview extension).
3.  A tab will open on the right. As you type your code, the preview will update automatically.


## 📝 Submission
To save your work to GitHub, follow these steps in the sidebar:

1.  Save your `index.html` file (`Ctrl+S` or `Cmd+S`).
2.  Go to the **Source Control** icon (the branch symbol on the far left).
3.  In the "Message" box, type: `Add week 02 html structure`.
